# task-management-application
This project is a Desktop-Based Task Scheduler developed in Python using Tkinter, integrated with a MySQL database to allow users to register, login, and manage tasks. The app supports task categorization, deadlines, priorities, and status tracking. Each task is linked to a user, making the system multi-user supported.

Aims and Objectives:
•	Provide a user-friendly interface for users to manage their daily tasks.
•	Allow users to categorize tasks with deadlines and priorities.
•	Integrate authentication (Login/Registration).
•	Maintain task logs and track task history.
•	Provide auto-update for task status after the deadline.



Project Scope:
The system is intended for:
•	Students managing academic and personal tasks.
•	Office workers tracking deadlines and deliverables.
•	Individuals maintaining to-do lists efficiently.


It supports:
•	User registration & login
•	Adding, editing, and deleting tasks
•	Categorizing and prioritizing tasks
•	Database-driven task storage
•	GUI-based interaction (Tkinter)
•	Task log/history tracking
